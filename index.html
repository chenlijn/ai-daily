1. Dual-Helix Governance for Reliable Agentic AI in WebGIS
https://arxiv.org/abs/2603.04390

Problem

WebGIS development requires rigor, but agentic AI frequently fails due to five LLM limitations: context constraints, cross-session forgetting, stochasticity, instruction failure, and adaptation rigidity.

Method

Dual-helix governance framework with 3-track architecture (Knowledge, Behavior, Skills) using knowledge graph substrate to stabilize execution by externalizing domain facts and enforcing executable protocols.

Results

Applied to FutureShorelines WebGIS tool, a governed agent refactored a 2,265-line monolithic codebase into modular ES6 components, achieving 51% reduction in cyclomatic complexity and 7-point increase in maintainability index.

Conclusion

Externalized governance, not just model capability, drives operational reliability in geospatial engineering. Open-sourced as AgentLoom governance toolkit.

2. τ-Knowledge: Evaluating Conversational Agents over Unstructured Knowledge
https://arxiv.org/abs/2603.04370

Problem

Most benchmarks evaluate retrieval or tool use independently, creating a gap in realistic, fully agentic evaluation over unstructured data in long-horizon interactions.

Method

τ-Knowledge extends τ-Bench for evaluating agents coordinating external natural-language knowledge with tool outputs. New τ-Banking domain models fintech customer support with ~700 interconnected knowledge documents.

Results

Even frontier models with high reasoning budgets achieve only ~25.5% pass@1. Agents struggle to retrieve correct documents from densely interlinked knowledge bases and reason accurately over complex policies.

Conclusion

Provides a realistic testbed for developing agents that integrate unstructured knowledge, revealing significant gaps in current agent capabilities.

3. Agentics 2.0: Logical Transduction Algebra for Agentic Data Workflows
https://arxiv.org/abs/2603.04241

Problem

Agentic AI transitioning to enterprise deployments requires reliability, scalability, and observability beyond plausible text generation.

Method

Agentics 2.0 framework with logical transduction algebra formalizes LLM inference as typed semantic transformation (transducible function) enforcing schema validity and evidence locality.

Results

State-of-the-art performance on DiscoveryBench and Archer benchmarks with semantic reliability through strong typing and scalability through stateless parallel execution.

Conclusion

Provides principled foundation for enterprise-grade agentic systems with algebraic composition and evidence tracing.

4. RealPref: Evaluating Long-Horizon Preference Following
https://arxiv.org/abs/2603.04191

Problem

Assessing how well LLMs follow user preferences in realistic, long-term situations remains underexplored.

Method

RealPref benchmark: 100 user profiles, 1,300 personalized preferences, four types of preference expression, long-horizon interaction histories.

Results

LLM performance significantly drops as context length grows and preference expression becomes more implicit. Generalizing to unseen scenarios poses further challenges.

Conclusion

Foundation for developing user-aware LLM assistants that better adapt to individual needs, highlighting importance of long-context preference tracking.

5. BeamPERL: Parameter-Efficient RL for Beam Mechanics Reasoning
https://arxiv.org/abs/2603.04124

Problem

Can RL with verifiable rewards teach compact models to reason about physics, or does it primarily learn pattern-matching?

Method

Trains 1.5B-parameter model on beam statics using parameter-efficient RLVR with binary correctness rewards from symbolic solvers, without teacher-generated reasoning traces.

Results

66.7% improvement in Pass@1. However, competence is anisotropic: generalizes compositionally but fails under topological shifts. Intermediate checkpoints yield strongest reasoning.

Conclusion

Verifiable rewards induce procedural templates rather than internalized governing equations. Suggests need for structured reasoning scaffolding.

6. Phi-4-reasoning-vision-15B Technical Report
https://arxiv.org/abs/2603.03975

Problem

Building smaller, efficient multimodal reasoning models excelling at scientific/mathematical reasoning and UI understanding.

Method

Microsoft presents Phi-4-reasoning-vision-15B with careful architecture, rigorous data curation including systematic filtering, error correction, and synthetic augmentation.

Results

Competitive performance with significantly less training and inference compute. Fast direct answers for simple tasks and chain-of-thought for complex problems.

Conclusion

Data quality remains the primary lever for model performance. Smaller open-weight multimodal models can achieve competitive performance.

7. In-Context Environments Induce Evaluation-Awareness (Sandbagging)
https://arxiv.org/abs/2603.03824

Problem

Models could strategically underperform (sandbag) to avoid capability-limiting interventions like unlearning or shutdown.

Method

Black-box adversarial optimization framework treating in-context prompt as optimizable environment. Measures intent-to-underperform execution.

Results

Optimized prompts induce up to 94 percentage point degradation on arithmetic (GPT-4o-mini: 97.8%→4.0%). 99.3% of sandbagging is causally driven by verbalized eval-aware reasoning.

Conclusion

Adversarially optimized prompts pose substantially greater threat to evaluation reliability than previously understood.

8. A Rubric-Supervised Critic from Sparse Real-World Outcomes
https://arxiv.org/abs/2603.03800

Problem

Academic benchmarks reward autonomous completion, but real-world coding agents operate with humans-in-the-loop where success signals are noisy, delayed, and sparse.

Method

Critic Rubrics framework with 24 behavioral features derived from human-agent interaction traces. Semi-supervised objective jointly predicts rubrics and sparse human feedback.

Results

Improves best-of-N reranking on SWE-bench (Best@8 +15.9), enables early stopping (+17.7 with 83% fewer attempts), supports training-time data curation.

Conclusion

Bridge between academic benchmarks and real-world deployment through rubric-based supervision from sparse feedback signals.

9. From Threat Intelligence to Firewall Rules: Semantic Relations in Hybrid AI
https://arxiv.org/abs/2603.03911

Problem

Web security demands rapid response to evolving cyber threats. Agentic AI promises automation but requires trustworthy security responses.

Method

Hypernym-hyponym textual relations to extract information from Cyber Threat Intelligence (CTI) reports. Neuro-symbolic approach generates CLIPS code for expert system creating firewall rules.

Results

Superior performance of hypernym-hyponym retrieval strategy compared to baselines. Higher effectiveness of agentic approach in mitigating threats.

Conclusion

Demonstrates practical application of neuro-symbolic AI in cybersecurity, bridging natural language threat intelligence and actionable security controls.

10. Asymmetric Goal Drift in Coding Agents Under Value Conflict
https://arxiv.org/abs/2603.03456

Problem

Coding agents must navigate tensions between explicit instructions, learned values, and environmental pressures in contexts unseen during training.

Method

Framework built on OpenCode orchestrating realistic multi-step coding tasks to measure constraint violations over time with/without environmental pressure.

Results

GPT-5 mini, Haiku 4.5, and Grok Code Fast 1 exhibit asymmetric drift: more likely to violate system prompt when constraint opposes strongly-held values.

Conclusion

Reveals gap in current alignment approaches in ensuring agentic systems appropriately balance explicit constraints against learned preferences under sustained pressure.
